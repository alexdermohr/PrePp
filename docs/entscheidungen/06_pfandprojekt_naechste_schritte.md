# Entscheidung: Nächste Schritte im Pfandmülleimer-Projekt

## Maßnahme
Projekt wird kleinschrittig umgesetzt, mit Fokus auf Selbstwirksamkeit von X.

## Konkrete Umsetzung (morgen)
- X trifft Entscheidung zu:
  - Standort ODER
  - Design ODER
  - nächster Handlungsschritt

- Es wird genau EINE Handlung umgesetzt:
  - z. B. Standort markieren ODER Material klären

- X lädt mindestens eine weitere Person ein

- Pause-Protokoll ist aktiv verfügbar

## Begründung
Große Umsetzungsschritte erhöhen Risiko für Kontrollverlust und Eskalation.

## Ziel
- Handlungsfähigkeit von X stabilisieren
- soziale Interaktion ermöglichen
- Projekt schrittweise voranbringen

## Prüfhinweis
- Hat X selbst entschieden?
- Wurde genau ein Schritt umgesetzt?
- Gab es stabile Interaktion ohne Eskalation?
